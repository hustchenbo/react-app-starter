# REACT APP STARTER

Starter template for react and webpack. Using the latest webpack, react-router and antd.

##feature

1. webpack v4
2. react-router v4
3. antd v3

##development mode

```bash
npm run dll

npm run server

```

visit http://localhost:3000/about


##production mode

```bash
npm run dll

npm run build

```
